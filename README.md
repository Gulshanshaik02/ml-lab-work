# ML Lab Work

This repository contains machine learning lab notebooks created for academic practice and learning purposes.

## Notebook
- `ml_lab_day1_full_workflow.ipynb`: Covers
  - Data Wrangling
  - Data Visualization
  - Logistic Regression
  - Decision Tree
  - Logistic Regression Interpretation
  - Node Identification and IF–THEN Rules

## Dataset
- `Customer_Purchase.csv`: Contains demographic and purchase behavior information for customers.
  - Columns include:
    - `Age` – Customer age
    - `Gender` – Male/Female
    - `Salary` – Monthly income
    - `Purchase` – YES/NO indicating purchase
- This dataset is used for all the ML experiments in the notebook.
